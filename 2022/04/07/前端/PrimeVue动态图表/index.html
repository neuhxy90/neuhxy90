PrimeVUE介绍

PrimeVue是Vue的丰富的开源组件集，集成了许多优秀的设计风格和灵活方便的组件库，相比于ElementUI和AntDesign具有更好的灵活性和现代化的展示效果。

知识盲区，查一下 Vue-setup组件选项。

setup()

使用 (data、computed、methods、watch) 组件选项来组织逻辑通常都很有效。然而，当我们的组件开始变得更大时，逻辑关注点的列表也会增长。尤其对于那些一开始没有编写这些组件的人来说，这会导致组件难以阅读和理解。

根据官方文档的介绍，当组件的逻辑较为复杂时，setup函数用来解决组合式API的。

setup 选项是一个接收 props 和 context 的函数，我们将在之后进行讨论。此外，我们将 setup 返回的所有内容都暴露给组件的其余部分 (计算属性、方法、生命周期钩子等等) 以及组件的模板。

使用setup函数有以下注意事项：

  • setup是使用Composition API 的入口
  • 在beforeCreate之前调用
  • 在setup中没有this
  • 返回对象中的属性刻在模板中使用，即在setup函数中定义的变量和方法最后都是需要 return 出去的 不然无法再模板中使用
