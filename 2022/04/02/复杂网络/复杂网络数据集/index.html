复杂网络数据集

发表于 2022-04-02 更新于 2022-09-08 分类于 复杂网络

在复杂网络的研究领域，数据集是非常重要的，下面就列举了一些常用的数据集获取的网站：

  1. Newman个人博客公布的数据集：http://www-personal.umich.edu/~mejn/netdata/
  2. Stanford Network Analysis Project 收集整理的数据集：http://snap.stanford.edu/index.html
  3. 斯坦福大学整理的数据集：https://web.stanford.edu/class/cs224w/resources.html
  4. 分类相似支持在线可视化的数据集网站：http://networkrepository.com/index.php
  5. 数据集很全的一个搜索入口：https://sparse.tamu.edu/
  6. 矩阵格式数据的一个网站（非常全）：http://konect.uni-koblenz.de/
  7. 数据挖掘领域的Aminer：https://www.aminer.cn/data-sna#Twitter-Dynamic-Net
  8. 有十几个社交类网络的数据集：https://toreopsahl.com/datasets/
  9. 科学网博客上的一篇介绍：http://blog.sciencenet.cn/blog-2952185-947972.html
  10. Pajek 的数据集：http://vladowiki.fmf.uni-lj.si/doku.php?id=pajek:data:urls:index
  11. Alex Arenas Website数据集：http://deim.urv.cat/~alexandre.arenas/data/welcome.htm
  12. 一些比较大的数据集或者KDD会议提供的数据集：http://iv.slis.indiana.edu/db/index.html
  13. 非常全面的一个数据集：http://moreno.ss.uci.edu/data.html
  14. 谷歌在线搜索数据集源：https://toolbox.google.com/datasetsearch
  15. 社交网络数据集：http://www.sociopatterns.org/datasets/
  16. Multilayer social network 数据集：http://multilayer.it.uu.se/datasets.html
  17. Multiplex Network 数据集：https://comunelab.fbk.eu/data.php

​ 最后，给出一个NEUSNCP平台数据实验，可以对 5Kb 以下的节点进行可视化分析，执行简单的社团划分算法、数据格式转换等：https://www.neusncp.com/api/datasets 。（不定期更新中）
